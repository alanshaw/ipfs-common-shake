# ipfs-common-shake

> Does [common-shake](https://github.com/indutny/common-shake) save some bytes when bundling IPFS?

Regular minified build:

```
$ npm run build
$ stat -f "%z" bundle.js
2572103
```

Minifed build + common-shake:

```
$ npm run build:shake
$ stat -f "%z" bundle.js
2516975
```

**Saving = 55,128 bytes**

You can `npm run serve` to run a static server and test the bundle works. It attached the IPFS node to `window.ipfs` for your testing purposes and should print "ready" to the console when the IPFS node is ready.
